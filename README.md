# 📰 Fake News / Misinformation Detector

An end-to-end **Data Science & NLP** project that automatically detects **Real** vs **Fake** news articles using Machine Learning.

## 📋 Project Overview
- **Dataset**: ISOT Fake News Dataset (~45,000 news articles)
- **Model**: TF-IDF Vectorizer + Logistic Regression
- **Accuracy**: ~98–99.5% on test data
- **Deployment**: Streamlit Community Cloud

## ✨ Features
- Paste any news headline or full article
- Instant prediction: **🟢 REAL NEWS** or **🔴 FAKE / MISINFORMATION**
- Confidence score with progress bar
- Clean and user-friendly interface
- Fast inference

## 🛠️ Tech Stack
- **Language**: Python
- **NLP & ML**: scikit-learn, NLTK, pandas
- **Deployment**: Streamlit
- **Model Saving**: joblib
