# Fake News / Misinformation Detector

End-to-end Data Science project that detects fake news using Machine Learning.

- Dataset: ISOT Fake News (~45k articles)
- Model: TF-IDF + Logistic Regression
- Accuracy: ~99%
- Deployed on Streamlit

## How to Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
